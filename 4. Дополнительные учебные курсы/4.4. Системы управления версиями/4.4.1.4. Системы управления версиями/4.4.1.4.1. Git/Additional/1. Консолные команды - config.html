Глобальные параметры "--global"

Указать глобальное имя пользователя - "user.name"

$ git config --global user.name "[имя пользователя]"

Пример:

$ git config --global user.name "Ivan Ivanov"

Указать глобальный e-mail - "user.email"

$ git config --global user.email [e-mail]

Пример:

$ git config --global user.email admin@siada.org

Указать текстовый редактор по умолчанию, который будет использоваться для ввода текста в Git - "core.editor"

$ git config --global core.editor [текстовый редактор]

Пример:

$ git config --global core.editor "C:\Program Files (x86)\Notepad++\notepad++.exe"

Указать программу, которая будет вызываться при возникновении конфликтов слияния кода - "merge.tool"

$ git config --global merge.tool [Программа]

Пример:

$ git config --global merge.tool "C:\Program Files (x86)\Meld\Meld.exe"

Просмотреть установленные параметры - "--list"

Пример:

$ git config --list
